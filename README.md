# Natursur

Aplicación web desarrollada con React y Vite, con backend en Supabase.

## Tecnologías

- React
- Vite
- JavaScript
- Supabase (Backend)

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado en tu sistema:

- [Node.js](https://nodejs.org/) (versión 16 o superior)
- npm (viene incluido con Node.js) o yarn

Para verificar que tienes Node.js instalado, ejecuta:
```bash
node --version
npm --version
```

##  Instalación y configuración local

Sigue estos pasos para levantar la aplicación en tu entorno local:

### 1. Clonar el repositorio
```bash
git clone https://github.com/JoseLu2121/natursur.git
cd natursur
```

### 2. Instalar dependencias
```bash
npm install
```

O si prefieres usar yarn:
```bash
yarn install
```

### 3. Ejecutar en modo desarrollo
```bash
npm run dev
```

O con yarn:
```bash
yarn dev
```

## Backend

El backend de la aplicación está desarrollado y desplegado en [Supabase](https://supabase.com/), que proporciona:

- Base de datos PostgreSQL
- Autenticación
- Storage para archivos
- API REST automática

No es necesario configurar ni desplegar el backend localmente, ya que está completamente operativo en la nube.
